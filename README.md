# lildengzi 的个人博客

基于 Astro + TailwindCSS 的个人博客网站。

![lildengzi-blog](https://github.com/user-attachments/assets/7a72aaae-6652-4cd0-becc-8e6a3c224993)

## 🔥 技术栈

- Astro
- TailwindCSS
- TypeScript

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview
```

## 📝 配置

在 `src/config/index.ts` 中修改个人信息。
